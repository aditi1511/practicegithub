git pull - will pull all the code from ur remote repo to ur workspace
git add - will add the file you have been working on 
git commit -m "write ur message here" - will write the commit message

git status is to see the status of the repo and to make sure everything is up to date

git reset will not add the thing you want to add, say you did it by accident
git reset commit will reset the commit you were about to do
